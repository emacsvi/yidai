
# 区块链算法内容概要


| 名称           | 描述                                 |
| :------------- | :----------------                    |
| 各种hash算法   | sha256, scrypt, 串联算法，并联算法等 |
| 共识算法       | 主要讲拜占庭问题与算法               |
| merkle树       | 比特币之中大量用到的默克尔树         |
| MPT树          | 帕特里夏树在以太坊之中的应用场景     |
| Bloom Filter   | 布隆过滤器在通信之中的应用           |
| 零知识证明     | zcash等零知识证明应用                |
| 多重签名       | 多重签名机制应用                     |
